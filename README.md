# Kinom Backend

Single backend API for web and mobile.

## Quick Start (Local)

1. Copy `.env.example` to `.env` and fill values.
2. Install deps: `npm install`
3. Run: `npm run dev`

## Docker

1. `docker compose up -d --build`
2. API: `http://localhost:5000/api/health` (or whatever `PORT` is set to)
